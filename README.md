# ICP Token Wallet

## Project Overview
A secure Rust-based token wallet for the Internet Computer Protocol blockchain.

## Prerequisites
- Rust (latest stable version)
- Cargo
- ICP SDK

## Setup
1. Clone the repository
2. Run `cargo build`
3. Run tests with `cargo test`

## Features
- Create wallets
- Transfer tokens
- Check token balances

## Running Tests
`cargo test`
